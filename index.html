This page demonstrates how CSS selectors work.

id

ids can be applied to only one item on a page

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div{
    background-color:yellow;
}

Here's what it looks like:

Inside my-div
class

Classes can be applied to more than one item on a page

Here's how we hook a class attribute to an element:

class="my-list"

Here's how the CSS is applied to the element:

li.my-list{
    background-color:orange;
}

compound

A compound selector allows more than one selector block to be separated by a comma

Here's how the CSS is applied to the element:

li.new-list,div#new-div{
    background-color:red;
}

descendant

A descendant selector effects all elements inside the current element, at any level

Here's how the CSS is applied to the element:

div.grandkids li{
    background-color:red;
}

child

A child selector allows us to access the next imediate elements inside the current element

Here's how the CSS is applied to the element:
